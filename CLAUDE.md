# EmbedStudio Developer Guide

## Commands
* **Build Project**: `npm run build`
* **Run Tests**: `npm run test`
* **Development Server**: `npm run dev`

## Deployment & Version Control Rules
* **Git Commit**: All modifications and fixes must be immediately committed locally with descriptive commit messages.
* **Git Push**: **CRITICAL** - Always push committed changes to the remote repository (`git push origin main`) after any modification or successful testing.
